# Grit Moments AI Lab
> AI를 잘 쓰기 위한 모임

매주 토요일, AI 설정 파일(CLAUDE.md, .cursorrules 등)을 직접 실험하고 개선하는 개발자 모임입니다

---
 
## 우리가 하는 일
 
같은 요구사항이라도 AI에게 어떤 컨텍스트를 주느냐에 따라 결과물이 완전히 달라집니다
 
우리는 매주 이 과정을 반복합니다
 
1. 이번 주 실험 주제를 정한다
2. 현재 MD 설정으로 AI에게 돌려서 **Before** 결과를 기록한다
3. 설정 파일을 수정한다 (규칙 추가, 컨텍스트 보강, 구조 변경)
4. 같은 요구사항으로 다시 돌려서 **After** 결과를 비교한다
5. 효과가 있으면 반영, 없으면 롤백
 
이 실험의 모든 과정은 히스토리로 남깁니다
